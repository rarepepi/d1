# **NoRampKit React SDK**

NoRamp empowers you to create a feature-rich and extensible checkout experience built on account abstraction, innovative payment flows, and much more. Enabling you to sell 70% more NFTs.

> These guides only covers **NoRampKit SDK.**

**Installation**

```jsx
npm i norampkit --save
# or
yarn add norampkit
```
